A.字符串之间的比较，是按照从左到右的顺序，逐位进行比较，按照unicode码的大小
charCodeAt(index) 用于获取指定索引值位置的unicode

B.减法运算 15-15=0 => false
C.隐式类型转换
‘21’ => 21 字符串转为数字
21 < 15 => false
D.’5’+0 => ‘50’ => true
+作用：
加法运算：左右两侧都是数字
字符串拼接：左右两侧只要有一侧是字符串，那么就是字符串拼接
